# Работа с транзакциями

Простой набор функций для сотрировки и фильтрации списка финансовых операций.

## Основные возможности

- **Фильтрация** транзакций по статусу (`state`)
- **Сортировка** транзакций по дате (`date`)

## Требования

- Python 3.6+
- Данные представлены gitв формате списка словарей (см. пример ниже)

## Установка

Скопируйте код функций в ваш проект или создайте файл `operations.py`.

## Доступные функции

### `filter_by_state(data, state='EXECUTED')`

Фильтрует транзакции по указанному статусу.

**Параметры:**
- `data` — список словарей с транзакциями
- `state` — значение статуса для фильтрации (по умолчанию `'EXECUTED'`)

**Возвращает:** новый список отфильтрованных транзакций


**Пример использования:**
```python
executed_tx = filter_by_state(transactions)  # Только EXECUTED
canceled_tx = filter_by_state(transactions, 'CANCELED')  # Только CANCELED
